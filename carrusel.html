Tutorial para crear un carrusel
Carrusel es un componente de JavaScript en Bootstrap que le permite agregar control deslizante a su sitio. Esto funciona en base a la transformación CSS 3D y le permite agregar diferentes variaciones de presentaciones de diapositivas.
El carrusel comienza con una etiqueta que tiene una ID única. En nuestro ejemplo, slider es la ID. El mismo debe incluir las clases '.carousel' y '.slide' y el atributo data-ride = 'carrusel'. Luego, la etiqueta div interna con '.carousel-inner' y el atributo role = 'listbox'. Luego otro div para cada elemento del carrusel con la clase '.carousel-item'. La etiqueta img tiene '.d-block' y '.img-fluid' para alinear las imágenes correctamente en el navegador.
Los indicadores deslizantes funcionan en función de la lista ordenada. La etiqueta usa una clase '.carousel-indicators' y cada etiqueta en la lista debe contener dos cosas: El atributo de destino de datos que debe tener el mismo valor que la identificación del carrusel principal. El atributo data-slide-to con el primer valor como 0 y aumentado a 1 y 2 para dos diapositivas adicionales.
Simplemente debe agregar un nuevo dentro de cada elemento del carrusel debajo de la etiqueta y usar la clase '.carousel-caption'. La clase '.d-none' se usa para ocultar los leyendas del carrusel en dispositivos móviles pequeños y traerlos de vuelta a los dispositivos medianos usando '.d-md-block'.
