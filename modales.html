Tutorial para crear una ventana modal

Los modales son simplificados, pero las instrucciones de diálogo flexibles funcionan con JavaScript. Son compatibles con una serie de casos de uso desde la notificación al usuario hasta el contenido completamente personalizado y cuentan con un puñado de subcomponentes útiles, tamaños y más.


En este caso utilice "Large Modal", el cual es un poco mas ancho que el "Small Modal", esto con el fin de darle comodidad al momento de visualizar el contenido.


Primero escribomos la siguiente linea:

(Para poder pegar el código evitaré abrir y cerrar las etiquetas, así que las reemplazaré con guines)


button - type="button" class="btn btn-primary" data-toggle="modal" data-target="#exampleModalLong"> Presione para ver tutorial - button


Ese botón es el que se mostrará dentro del card y al darle click llamará con data-toggle="modal" data-target="#exampleModalLong" a la ventana modal que crearemos a continuación.


div class="modal fade bd-example-modal-lg" id="exampleModalLong" tabindex="-1" role="dialog" aria-labelledby="exampleModalLongTitle" aria-hidden="true"


Como observamos, con la propiedad bd-example-modal-lg" extendemos el ancho de nuestra ventana modal y posteriormente la creamos.


Dentro de "div class="modal-content"" irá el contenido de nuestra ventana modal.
